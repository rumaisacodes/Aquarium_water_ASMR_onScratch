# 🐟 Aquarium Water - Peaceful ASMR

An interactive, ambient aquarium ASMR experience built using Scratch and embedded directly into a clean HTML webpage. Perfect for relaxing, studying, or leaving on in the background as a calming screensaver.


---

## 🌊 About the Project

This project combines the visual creativity of **Scratch** with the accessibility of **web development**. I wanted to create a peaceful, looping aquarium environment with soothing water sounds and gentle fish movements. 

By exporting and embedding the Scratch project into an HTML structure, this repository provides a standalone, distraction-free web page dedicated entirely to relaxation.

### ✨ Features
* **Ambient Audio:** Calming water bubbles and gentle aquatic background sounds.
* **Fluid Visuals:** Hand-coded Scratch animations featuring realistic fish behaviors and floating bubbles.
* **Distraction-Free UI:** A minimalist HTML container that focuses entirely on the ASMR experience.
* **Lightweight:** Runs smoothly right in your web browser with no heavy installations required.

---

## 🛠️ Built With

* **[Scratch](https://scratch.mit.edu/):** Used for the logic, animations, and sound triggers.
* **HTML5 / CSS3:** Used to structure the webpage, center the project, and style the ambient background.

---


## 📝 Code Overview

The project is structured simply for easy modification:
* `index.html` - The main entry point containing the embedded Scratch player iframe and container styles.
* `README.md` - Project documentation.

> **Note on Scratch Embedding:** The Scratch project is rendered via an iframe using the standard Scratch embed format or an open-source player like TurboWarp for enhanced performance.
